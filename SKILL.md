# 小红书亲子地点挖掘 (xiaohongshu-spot-finder)

## 核心功能

**从小红书搜索符合要求的亲子地点 → 自动添加到飞书多维表格**

## 触发条件

当用户：
- 说"去小红书找找新的亲子地点"
- 问"最近杭州有什么新开的遛娃地方"
- 说"更新一下亲子表格"
- 提到"小红书推荐"、"网红打卡地"等关键词

## 使用方法

### 基础用法
```
去小红书找点新的亲子游地方
```

### 指定季节/主题
```
找一下春天适合赏花的地方
找一下夏天室内玩水的地点
```

### 指定区域
```
找一下余杭区的新地方
找一下未来科技城附近的
```

## 完整工作流程

```
1. 搜索 🔍
   └─ 用 browser 或 web_search 搜索小红书相关内容
   └─ 关键词：杭州 + 亲子游 + [季节/主题] + 小红书

2. 解析 📝
   └─ 从搜索结果提取地点信息
   └─ 字段：地点名、区域、特色、适合年龄等

3. 去重 ✅
   └─ 读取飞书表格现有记录
   └─ 检查地点名称是否已存在
   └─ 跳过重复的，只添加新的

4. 同步 📊
   └─ 调用 feishu_bitable_create_record
   └─ 填充所有字段（区域、标签、推荐理由等）

5. 汇报 📢
   └─ 告诉用户添加了哪些地方
   └─ 哪些是重复的已跳过
   └─ 给出本周推荐建议
```

## 搜索策略

### 季节性关键词
- 春季：樱花、郁金香、赏花、踏青、放风筝
- 夏季：玩水、室内、空调、避暑、游泳
- 秋季：桂花、登山、银杏、采摘
- 冬季：室内、温泉、滑雪、看展

### 热门主题
- 网红打卡、拍照出片、小众秘境
- 免费、性价比高、遛娃神器
- 大草坪、野餐、帐篷

## 输出格式

```
📕 从小红书找到 X 个新地点：

✅ 已添加：
1. [地点名称] - [区域] ⭐[推荐指数]
   特色：[标签]
   理由：[简短推荐理由]

⚠️ 已存在（跳过）：
1. [地点名称] - 表格里已经有了

💡 建议：
- 本周推荐去 [地点]，因为 [原因]
```
