# 小红书亲子地点挖掘 (xiaohongshu-spot-finder)

## 触发条件

当用户：
- 说"去小红书找找新的亲子地点"
- 问"最近杭州有什么新开的遛娃地方"
- 说"更新一下亲子表格"
- 提到"小红书推荐"、"网红打卡地"等关键词

## 使用方法

### 基础用法
```
去小红书找点新的亲子游地方
```

### 指定季节/主题
```
找一下春天适合赏花的地方
找一下夏天室内玩水的地点
```

### 指定区域
```
找一下余杭区的新地方
找一下未来科技城附近的
```

## 工作流程

1. **搜索** - 用 web_search 搜索小红书相关内容
2. **解析** - 提取地点名称、区域、特色等信息
3. **去重** - 检查飞书表格，避免重复添加
4. **添加** - 将新地点添加到飞书多维表格
5. **汇报** - 告诉用户添加了哪些地方

## 搜索策略

### 季节性关键词
- 春季：樱花、郁金香、赏花、踏青、放风筝
- 夏季：玩水、室内、空调、避暑、游泳
- 秋季：桂花、登山、银杏、采摘
- 冬季：室内、温泉、滑雪、看展

### 热门主题
- 网红打卡、拍照出片、小众秘境
- 免费、性价比高、遛娃神器
- 大草坪、野餐、帐篷

## 输出格式

```

✅ 已添加：

1. [地点名称] - 表格里已经有了
```
