# 📕 小红书亲子地点挖掘 (xiaohongshu-spot-finder)

从小红书自动搜索杭州亲子游玩地点，并添加到飞书多维表格。

## 功能特点

- 🔍 **智能搜索** - 根据季节、主题、区域搜索小红书热门地点
- ✅ **自动去重** - 检查飞书表格，避免重复添加
- 📊 **结构化存储** - 自动填充区域、适合年龄、特色标签等字段
- 🌸 **季节推荐** - 根据当前季节调整搜索关键词

## 使用方法

### 在 OpenClaw 中直接使用

```
去小红书找点新的亲子游地方
```

```
找一下春天适合赏花的地方
```

```
找一下余杭区的新地方
```

### 搜索策略

| 季节 | 关键词 |
|------|--------|
| 春季 | 樱花、郁金香、赏花、踏青、放风筝 |
| 夏季 | 玩水、室内、空调、避暑、游泳 |
| 秋季 | 桂花、登山、银杏、采摘 |
| 冬季 | 室内、温泉、滑雪、看展 |

## 技术实现

- **搜索**：使用 Brave Search API 搜索小红书相关内容
- **解析**：从搜索结果提取地点信息
- **存储**：通过 Feishu Bitable API 添加到飞书表格
